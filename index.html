How it works

Get started in three steps:

  1. Under Load JSON, pick Power Apps export JSON for Version A (and Version B if you want to compare). Use Load Sample for a quick bundled demo.
  2. Explore formulas with Dependency, Tree View, Logic Flow, or Code View. Click a node or formula to see details in the inspector.
  3. With two versions loaded, press Compare Versions and review results in Version Diff.
